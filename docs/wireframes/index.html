Fraction App — Room Wireframes

Spatial wireframes for every level of the fraction-adding tutoring game, in the wireframe Skin (boxes + system font, no art). Each page shows that room's lead scene on the logical 1280×800 stage plus a beat strip of how the space evolves. Layout follows the scene-region grammar (presentation-scene-architecture §4–§5, §10); the goal line is voiced and shown on screen as text. Swap in the Kitchen Skin later and the layout is unchanged — only the look.

Counting Room

A written number is how many objects there are. Place pieces, watch the tally climb, write the numeral. Terminal: the mixed scatter.

Tidying Room

R4 · IMPROPER→MIXED

Whole-Units Room

Nine sevenths. Group an overflow stack into wholes against the ruler; read off the mixed number (wholes + leftover).

Order of play: kitchen hub → R0a → R0b → R1 → R2 → R3 → R4, with diagnosis routing children to the room they need. Every room except R0a ends on the shared bare slate (write the answer by hand). Wireframes are spatial only — final art arrives as a second Skin without moving anything.
