On the Number Line
0 1 ¼ 2/4 ¾ 34
0 1
A fraction is a number — one point on the line.
Cut the line from 0 to 1 into four equal parts.
Slide the red point three parts along…
The fill grows to show three fourths of the way.
That point is the number three fourths.
So where does three fourths come from? We add.
Start with two fourths — two equal pieces.
Draw a second line: one more fourth, the same size.
Same bottom, same-size pieces — slide them together.
2/4 + 1/4 = 3/4 — keep the bottom, add the tops.
